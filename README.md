# Habit Tracker (React + Vite)

## Run locally
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
1. Edit `vite.config.js` and replace `REPO_NAME_HERE` with your GitHub repo name.
2. On GitHub: Settings → Pages → Source: **GitHub Actions**
3. Push to `main`. Your site will deploy to:
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
